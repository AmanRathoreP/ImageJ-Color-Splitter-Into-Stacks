<p align="center">
  This plugin splits an image or stack into multiple color spaces (RGB, CMYK, CIELAB, HSV) and displays each channel as a grayscale image.
</p>


## Features

* Works with ImageJ and Fiji
* Supports splitting into multiple color spaces at once
* Easy to use
* Free and open source

## Quick start

### Via ImageJ official website
- Step 1.) Download the latest version(.class) of the addon from the ImageJ website
- Step 2.) Go to ImageJ and then Plugins > Install
- Step 3.) Select the `Color_Splitter_Into_Stacks.class`
> NOTE: This is the [ImageJ homepage]() for the plugin.
### Via custom download
- Step 1.) Download the latest version(.class) of the addon from the [release section]()
- Step 2.) Go to ImageJ and then Plugins > Install
- Step 3.) Select the `Color_Splitter_Into_Stacks.class`

## Contributing [![contributions welcome](https://img.shields.io/badge/contributions-welcome-brightgreen.svg?style=flat)](issues.md)

Thank you for considering contributing to ImageJ Color splitter!

First note we have a code of conduct, please follow it in all your interactions with the program files.

We welcome any type of contribution, _not only code_. You can help with:
- **QA**: File bug reports, the more details you can give the better (e.g. images or videos)
- **New Features**: You can suggest an modifications or just ask for advancements in the old features of the plugin.
- **Code**: Take a look at the [open issues](issues.md). Even if you can't write the code yourself, you can comment on them, showing that you care about a given issue matters. It helps us to handel them.
